
# 🚀 Telegram 频道强制订阅与免后台动态管理机器人 (菜单隔离安全版)

这是一个基于 **Cloudflare Workers + KV** 架构开发的高性能、零成本 Telegram 自动回复机器人。专门用于**“订阅频道后才能获取节点/机密信息”**的业务场景。

本版本加入了 **动态身份菜单隔离引擎**。普通用户和管理员在手机端看到的左下角内置菜单完全不同：普通用户只能看到基础查询选项；管理员则会自动附加独占的 `/manage` 控制台选项，完美隐藏管理路径。

### 🌟 核心特性
* 🔒 **死锁拦截级强力防绕过**：采用实时线上接口校验，用户关注后取消订阅或秒退频道，机器人下一秒立刻封锁，绝无白嫖可能。
* 🛡️ **智能全球分布式防刷熔断**：消息刚进系统立刻经由 CF 骨干网络 KV 锁鉴定，一秒内超过 2 次的恶意超频请求在入口处直接物理抹杀，绝不向后执行，彻底保护你的 Workers 免费额度。
* ⏱️ **精准内存自动防灾**：内置双轨降级限流引擎，若未绑定 KV 时会自动采用局部内存锁，且请求处理完 3 秒内物理强制自毁，绝不堆积内存，防范 Workers 128MB 限制引起的崩溃重启。
* 🎛️ **动态身份菜单隔离 (Scope Commands)**：彻底弃用全局菜单。机器人会根据交互者的电报 ID 实时对手机端菜单进行沙箱重构。管理员可见专属管理面板，普通用户完全隐性无感知。
* 🤝 **人性化客户引导**：内置小白式使用指南、点击自动复制代码块、并直达内置双向客服私聊机器人。

## ⚙️ 第二部分：核心环境变量配置明细

保存代码后，请前往 Worker 的 **`Settings` (设置) -> `Variables` (变量)** 页面，点击 **`Add`** 依次配置以下 **5个** 核心环境变量：

| 变量名称 (Key) | 作用描述 | 填写示例格式 |
| :--- | :--- | :--- |
| **`BOT_TOKEN`** | 你的 Telegram 自动回复机器人 Token | `123456789:ABC...（从@BotFather获取）` |
| **`CHANNEL_ID`** | 你的官方公网频道用户名（必须带 `@` 符号） | `@xxxx` |
| **`CHANNEL_LINK`** | 拦截用户时，引导跳转订阅频道的链接 | `https://t.me/xxxx` |
| **`ADMIN_ID`** | 老板本人的 Telegram 纯数字账号 ID | `123456789` *(私聊@username_to_id_bot可查)* |
| **`NODE_RULES`** | *（选填）* 首次上线的纯文本节点备用库 | *KV为空时自动以此兜底，格式：关键词===内容* |

## 🚀 第三部分：高级进阶——启用全球分布式 KV 空间绑定

### 1. 创建 KV 命名空间
1. 登录 Cloudflare 控制台，进入 **`KV`** 页面。
2. 点击右上角 **`Create a namespace`**，为其命名（例如：`TG_BOT_DB`），点击 **`Add`**。

### 2. 将 KV 绑定至机器人 Worker
1. 进入你机器人的 Worker 管理页面，点击 **`Settings` (设置)** -> 选择左侧的 **`Bindings` (绑定)**。
2. 在 **KV Namespace Bindings** 区域点击 **`Add Binding`**。
3. **严格按照以下规范填写**：
   * **变量名称 (Variable Name)**：必须手动输入大写的 **`TG_LIMIT_KV`**
   * **KV 命名空间 (KV Namespace)**：在下拉菜单中选中刚才创建的 `TG_BOT_DB`。
4. 点击最下方的 **`Save and Deploy`** 即可。

## 🚀 第四5部分：保姆级零瑕疵部署与动态激活

### 步骤 1：让 Bot 成为频道管理员
1. 打开 Telegram，将机器人拉入频道并设置为管理员，保持默认权限开启并点击右下角 **`保存`**。

### 步骤 2：部署代码与绑定变量
1. 登录 Cloudflare 将第一部分超链接中的代码全选覆盖进你的 Worker，并在后台绑定好第二、第三部分要求的 5 个环境变量与大写 `TG_LIMIT_KV` 空间。

### 步骤 3：激活 Webhook 通道（仅需执行一次）
将以下链接中的 `<你的BOT_TOKEN>` 和 `<你的Worker独立域名>` 替换为你自己的真实数据，然后复制到浏览器中访问以激活通道：
```text
[https://api.telegram.org/bot](https://api.telegram.org/bot)<你的BOT_TOKEN>/setWebhook?url=https://<你的Worker独立域名>/

```

### 步骤 4：激活动态菜单显示

1. 部署完成后，老板本人直接前往你的机器人私聊界面，发送一次 `/start` 触发身份探测。
2. 机器人判断你是管理员后，你手机左下角的 `Menu` 菜单会自动激活并加载 `/start`、`/help`、`/manage` 三个完整指令。
3. 当普通用户私聊发消息时，系统会自动对其隐藏 `/manage` 控制台，仅保留最基础的安全指令。

## 📱 第五部分：老板专属快捷管理实操说明

管理员直接在聊天框点击左下角菜单中的 `/manage` 即可唤出最高权限控制台：

### 1. 快捷添加/更新某个分类节点

1. 点击控制台下发的 **[1. 快捷添加/更新节点模板]** 对应的那行蓝色代码块，手机端会**自动复制整行暗号**。
2. 粘贴到输入框中，将 `香港节点` 改为你想要的关键词，把后面的 `vmess://...` 换成你真实最新的节点链接。
3. 点击发送。系统将自动分类并永久存储进分布式主干网，全网实时秒级生效。

### 2. 快捷彻底下架/删除某个分类

1. 点击控制台下发的 **[2. 快捷下架/删除节点模板]** 对应的那行代码块自动复制。
2. 粘贴到输入框中，把里面的 `香港节点` 替换为你想要消灭的触发词，直接发送即可秒级全网下架。
