
# 🚀 Telegram 频道强制订阅与免后台可视化管理系统 (工业级生产完满版)

这是一个基于 **Cloudflare Workers + 边缘内存双轨沙盒 + CF-KV** 架构开发的高性能、零成本、超低延迟 Telegram 自动回复引流系统。专门用于**“订阅公网频道后才能私聊提取/解锁节点、私密网站、纯文字公告或海报大图”**的私域闭环运营场景。

本系统历经多轮大厂生产级压测打磨，在底层攻防、高并发、资源节约及内存泄漏控制上进行了极致微操调优，是真正意义上的“免后台、可视化、小白式”钢铁核心系统。

## 💎 核心技术成就 (生产环境四大命门加固)

* 🛡️ **全球分布式网络防灾与 8 秒硬熔断**：前置挂载 `AbortController` 8秒硬时钟控制器。当 Telegram 官方网络或跨国骨干网局部发生严重卡顿抖动时，Worker 能够在 8 秒内强行自我切断并优雅退出，彻底根治传统脚本在 Serverless 场景下被僵尸连接活活卡死直至超时的灾难。
* 📉 **成功/失败双轨高速 TTL 缓存沙盒**：针对高频身份接口 `getChatMember` 引入絕對 TTL 机制。
  * **订阅成功者**：在内存中被冻结 **60 秒**，在 1 分钟内疯狂获取不同节点，响应延迟全部为 **0 毫秒**，彻底斩断对 Telegram API 的无谓浪费。
  * **未订阅者（失败）**：强行降级缓存生命周期，使其**仅活 5 秒钟**。既在入口处完美抵挡了黑客机器脚本的超频压测，又给真实粉丝在点完“立即订阅”折返回来时留下了 5 秒黄金真空期，发词即秒过，杜绝死锁差评。
* 🔒 **单用户单定时器独占与内存防泄漏**：`menuCache`（菜单自净锁，24小时到期）与 `subCache` 均引入了点对点定时器前置清除机制（`clearTimeout`）。在高并发和长周期的持续流量冲击下，任何一个用户在 Worker 内存中永远仅仅独占唯一一个定时器指针，**100% 杜绝了高日活下 Event Loop 垃圾指针堆积引发的 128MB 内存泄漏崩溃**。
* 👑 **全局 Bot Username 永久常驻缓存**：全系统对 `getMe` 接口实施单容器仅抓取一次策略，抓取后永久封存，消灭了每次添加规则或老板打开 `/manage` 控制台时由于重复握手产生的 API 开销。
* 🚫 **HTML 属性防逃逸与特殊字符防爆**：引入全量 `escapeHtml` 机制。更在超链接属性插槽内部并入了双引号暴力百分号化（`replace(/"/g, "%22")`）以及国际标准的 `encodeURI` 编码。黑客或管理员在误添加带有双引号、`<`、`>`、`&` 的长网址或特殊名字时，超链接的引号墙永远稳固，**彻底终结了 `Bad Request: can't parse entities` 的解析爆裂隐患**。

## 🚀 第二部分：高级进阶——启用全球分布式 KV 空间绑定

为了激活免后台动态增删节点和数字删除面板，必须为 Worker 绑定 KV 分布式数据库空间。

### 1. 创建 KV 命名空间
1. 登录 Cloudflare 控制台，选择左侧导航栏的 **`KV`** 页面。
2. 点击右上角 **`Create a namespace`**，为其命名（推荐输入：`TG_BOT_DB`），点击 **`Add`**。

### 2. 将 KV 绑定至机器人 Worker
1. 进入你机器人的 Worker 管理主页面，点击 **`Settings` (设置)** -> 选择左侧的 **`Bindings` (绑定)**。
2. 在 **KV Namespace Bindings** 区域点击右侧的 **`Add Binding`**。
3. **严格按照以下工业规范填写**：
   * **变量名称 (Variable Name)**：必须手动输入全大写的：**`TG_LIMIT_KV`**
   * **KV 命名空间 (KV Namespace)**：在下拉菜单中选中刚才创建的 `TG_BOT_DB`。
4. 点击最下方的 **`Save and Deploy` (保存并部署)** 即可。

2. 保持默认的发布消息权限开启，点击右下角 **`保存`**。

### 步骤 2：激活 Webhook 网络通道（仅需执行一次）
将以下链接中的 `<你的BOT_TOKEN>` 和 `<你的Worker独立域名>` 替换为你的真实变量数据，然后复制到浏览器中敲回车访问。当页面返回 `{"ok":true,"description":"Webhook was set"}` 时，说明网络通道全面打通：
```text
[https://api.telegram.org/bot](https://api.telegram.org/bot)<你的BOT_TOKEN>/setWebhook?url=https://<你的Worker独立域名>/

```

1. 部署完成后，老板本人直接前往你的机器人私聊界面，发送一次 `/start` 激活身份探针。
2. 机器人通过电报服务器的数字签名确认你是最高管理员后，你手机左下角的 `Menu` 菜单会自动重构加载 `/start`、`/help`、`/manage` 三个完整指令。
3. 普通用户无论怎么刷，他们手机左下角只会显现基础的 2 个用户选项，管理后台对他们完全隐性、宇宙隔离，**绝无任何串错可能**。

## 📱 第四部分：老板专属后台管理精细使用指南

老板日常维护完全**不需要打开 Cloudflare 后台**，直接在机器人的私聊框点击左下角菜单中的 `/manage` 即可一键拉出当前 KV 的全部数据列表账单。

上面的灰色代码块都具有电报原生特性，**用手指在手机屏幕上点一下代码块，它就自动复制到了你的剪贴板**，修改内容后发出去就能百分之百闭环运行：

### 1️⃣ 快捷添加/更新【文字节点】

格式：`添加#关键词#频道显示的脱敏备注#用户私聊拿到的真实内容`

```text
添加#香港节点#香港专线#🇭🇰 <b>香港专线节点已更新</b>\n\n<code>vmess://xxxxxx</code>

```

* **公网频道表现**：刚刚为您上新/优化了 **香港节点** 相关的专属获取规则！
* **脱敏防灾机制**：公网频道广播出去的文本干净干净，敏感节点数据、内部私密网站在这一层完全加密，完美保护资产。

### 2️⃣ 快捷添加/更新【网页网址/群组绿色遮罩引流】

格式：`添加#真实网址#频道显示的绿色遮罩备注#用户私聊拿到的文本`

```text
添加#[https://t.me/your_qun#](https://t.me/your_qun#)💬 点击加入技术交流群#欢迎加入官方群组交流！

```

* **公网频道表现**：长网址被彻底隐形！公网频道里发出的广告会完美渲染为绿色的超链接：[💬 点击加入技术交流群]，粉丝点击即可跳转！

### 3️⃣ 快捷添加/更新【纯文字通知公告】

格式中的第一个参数直接借用老板机器人自己的私聊链接，即可产生奇效：

```text
添加#[https://t.me/你的机器人名字#](https://t.me/你的机器人名字#)📢 官方全新升级公告#感谢大家支持，本系统已完成高并发架构升级！

```

* **公网频道表现**：在公网频道里发出一张干净利落的纯文字蓝绿超链接标题：[📢 官方全新升级公告]，用户一点击就能引流直达你的机器人私聊解锁！

### 4️⃣ 快捷添加/更新【精美海报大图广告引流】

1. 先把你做好的活动海报图片发在任意公开频道或群组，右键选择 **“复制消息链接 (Copy Post Link)”**，得到一个电报官方图片网络链接（如 `https://t.me/xqkin/123`）。
2. 将此链接作为网址参数输入指令：

```text
添加#[https://t.me/xqkin/123#](https://t.me/xqkin/123#)🖼️ 查看活动海报详情#🎁 欢迎参加特惠活动！优惠码：LUCK666

```

* **公网频道表现**：频道发出的文字超链接下方，**会利用 Telegram 官方的链接预览机制，自动展开海报的预览大图**！完美实现零额外内存消耗的图文并茂广告引流。

### 5️⃣ 🔍 规则盘点与精准选择性数字删除

1. 随时在聊天框发送 `/manage` 呼出控制后台。
2. 机器人会弹出一张精致的 **HTML 实时账单**，将数据库里的每一条规则，按照从上到下的顺序，用代码块包裹，并强行生成前面的 `1.` `2.` `3.` 纯数字编号。
3. 看着账单，看哪一条不顺眼（例如第 1 条），直接发送暗号：
```text
删除#1

```


4. 系统会在后台瞬间切除该规则并重新洗牌写入 KV。再次发送 `/manage` 即可看到账单已被彻底剔除更新。
5. 同时系统也支持通过关键词直接匹配删除：
```text
删除#香港节点

```



### 6️⃣ 🚫 工业级动态黑名单拦截管控

* **一键拉黑恶意刷屏账户**：发送 `拉黑#用户纯数字ID`（例如：`拉黑#6543210`），被拉黑者将瞬间在系统最外层防线被全面封锁。当他们再次发词时，机器人会冷酷回复：“访问受限...”。
* **一键解除封禁**：发送 `解黑#用户纯数字ID`（例如：`解黑#6543210`），即可重新释放用户，恢复其获取节点的权限。
