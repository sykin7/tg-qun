
# 🚀 Telegram 频道强制订阅与免后台可视化管理机器人 (数字精准删除版)

这是一个基于 **Cloudflare Workers + KV** 架构开发的高性能、零成本 Telegram 自动回复引流机器人。专门用于**“订阅频道后才能获取节点/广告/网站链接/纯文字公告”**的私域闭环运营场景。

本版本完美引入了 **15秒局部内存双重缓存拦截机制**、**自定义通道链接/文字遮罩**、**后台数字编号账单可视化盘点** 以及 **按编号精准选择性删除引擎**。在极度节省 KV 免费额度的同时，提供了工业级的抗揍防御和绝佳的免后台管控体验。

### 🌟 核心特性
* 🔒 **死锁拦截级强力防绕过**：采用实时线上接口强校验，用户关注频道拿完线路后若秒退或取消关注，机器人下一秒立刻封锁拦截。
* 🛡️ **智能全球分布式防刷熔断**：请求入境立刻经由局部锁与 KV 锁双重鉴定，一秒内超过 2 次的恶意超频请求在入口处直接物理隔离，彻底保护 Workers 免费额度。
* 📉 **15秒局部内存缓存（极度省额度）**：将黑名单与规则库自动在 Worker 实例中缓存 15 秒，避免高并发下频繁查询 KV。发广播及频道推送走纯官方网络通道，**对 KV 的额外消耗为完美的零次**。
* 🎛️ **动态身份菜单隔离 (Scope Commands)**：彻底弃用全局菜单。机器人会根据交互者的电报 ID 实时对手机端菜单进行沙箱重构。管理员可见专属管理面板，普通用户完全隐性无感知，**绝无串错可能**。
* 🔍 **KV 库一键可视化账单明细**：管理员发送 `/manage`，机器人会自动拉出当前数据库存的所有规则，并强行生成 `1. 2. 3.` 纯数字编号代码块，里面存了什么一目了然。
* 🗑️ **编号精准选择性删除**：彻底弃用输入长网址删除的臃肿方式！支持直接发送 `删除#数字编号`（如 `删除#2`）精准切除规则，全网瞬间刷新生效。

## 🚀 第二部分：高级进阶——启用全球分布式 KV 空间绑定

### 1. 创建 KV 命名空间
1. 登录 Cloudflare 控制台，进入 **`KV`** 页面。
2. 点击右上角 **`Create a namespace`**，为其命名（例如：`TG_BOT_DB`），点击 **`Add`**。

### 2. 将 KV 绑定至机器人 Worker
1. 进入你机器人的 Worker 管理页面，点击 **`Settings` (设置)** -> 选择左侧的 **`Bindings` (绑定)**。
2. 在 **KV Namespace Bindings** 区域点击 **`Add Binding`**。
3. **严格按照以下规范填写**：
   * **变量名称 (Variable Name)**：必须手动输入大写的 **`TG_LIMIT_KV`**
   * **KV 命名空间 (KV Namespace)**：在下拉菜单中选中刚才创建的 `TG_BOT_DB`。
4. 点击最下方的 **`Save and Deploy`** 即可。

## 🚀 第三部分：保姆级零瑕疵部署与动态激活

### 步骤 1：让 Bot 成为频道管理员
1. 打开 Telegram，将机器人拉入频道并设置为管理员，保持默认权限开启并点击右下角 **`保存`**。

### 步骤 2：激活 Webhook 通道（仅需执行一次）
将以下链接中的 `<你的BOT_TOKEN>` 和 `<你的Worker独立域名>` 替换为你的真实数据，然后复制到浏览器中访问以激活通道：
```text
[https://api.telegram.org/bot](https://api.telegram.org/bot)<你的BOT_TOKEN>/setWebhook?url=https://<你的Worker独立域名>/

```

### 步骤 3：激活动态菜单显示

1. 部署完成后，老板本人直接前往你的机器人私聊界面，发送一次 `/start` 触发身份探测。
2. 机器人判断你是管理员后，你手机左下角的 `Menu` 菜单会自动重构并加载 `/start`、`/help`、`/manage` 三个完整指令。普通用户端会自动隐藏管理选项。

## 📱 第四部分：老板专属可视化管理实操指令明细

管理员直接在私聊框点击左下角菜单中的 `/manage` 即可一键拉出当前 KV 的全部数据列表账单。

### 1. 普通文字节点/文本添加

格式：`添加#关键词#频道显示的脱敏备注#用户私聊拿到的节点回复文本`

```text
添加#香港节点#🇭🇰 香港专线#🇭🇰 **香港专线节点已更新**\n\`vmess://xxxx\`

```

* **频道广播表现**：公网频道会直接干净地发出加粗字体：**香港节点**，敏感节点数据完全脱敏隐藏。

### 2. 网站链接/群组/广告完美遮罩引流

格式：`添加#真实网址#频道显示的绿色遮罩备注#用户私聊拿到的文本`

```text
添加#[https://t.me/your_qun#](https://t.me/your_qun#)💬 点击加入技术交流群#欢迎加入官方群组交流！

```

* **频道广播表现**：公网频道会隐藏长网址，将其作为底层数据打包进绿色的超链接中，界面完美呈现为：[💬 点击加入技术交流群]，点击即可跳转！

### 3. 广播纯文字公告/广告玩法

既然第一个参数必须写网址才能触发频道遮罩，我们直接把**你自己的机器人私聊网址**作为第一个参数：

```text
添加#[https://t.me/your_bot#](https://t.me/your_bot#)📢 官方全新升级公告#感谢大家支持，本系统已完成高并发架构升级！

```

* **频道广播表现**：频道内网址被完美隐藏，显示为漂亮的纯文字蓝绿标题：[📢 官方全新升级公告]，用户点击不仅能看公告，还能直接一键引流跳进机器人私聊。

### 4. 高级海报大图广告玩法（图文外链法）

1. 先把你做好的广告海报图片发在任意公开频道或群组，右键选择 **“复制消息链接 (Copy Post Link)”**，得到一个电报官方图片链接（如 `https://t.me/xqkin/123`）。
2. 将此链接作为网址参数输入指令：

```text
添加#[https://t.me/xqkin/123#](https://t.me/xqkin/123#)🖼️ 查看活动海报详情#🎁 欢迎参加特惠活动！优惠码：\`LUCK666\`

```

* **频道广播表现**：频道中会完美显示绿色遮罩文本，且**由于 Telegram 官方的链接预览机制，通知下方会自动展开你那张精美的海报大图预览！** 完美实现零内存消耗的图文广告引流。

### 5. 🔍 账单盘点与 Precision 精准选择性删除

1. 发送 `/manage` 呼出后台，在规则明细列表中，每一条规则前都有一个数字（如 `1.` `2.` `3.`）。
2. 看哪一条不顺眼，直接发送暗号：`删除#数字编号`。
* 例如你想删除第 2 条群组链接规则，直接发送：
`删除#2`


3. 系统会在后台瞬间切除该规则并重新写入 KV。再次发送 `/manage` 即可看到规则已成功被剔除。

* **一键解除封禁**：发送 `解黑#用户数字ID`（例如：`解黑#6543210`），即可重新释放用户。
