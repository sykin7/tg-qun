
# 🚀 Telegram 频道强制订阅与免后台动态管理机器人 (自由遮罩防刷版)

这是一个基于 **Cloudflare Workers + KV** 架构开发的高性能、零成本 Telegram 自动回复引流机器人。专门用于**“订阅频道后才能获取节点/广告/网站链接”**的私域闭环运营场景。

本版本引入了 **局部内存双重缓存拦截机制**、**自定义通道链接富文本遮罩** 以及 **沙箱隔离级多重动态菜单**，在极度节省 KV 免费额度的同时，提供了工业级的抗揍防御和绝佳的防白嫖引流体验。

### 🌟 核心特性
* 🔒 **死锁拦截级强力防绕过**：采用实时线上接口强校验，用户关注频道拿完线路后若秒退或取消关注，机器人下一秒立刻封锁拦截。
* 🛡️ **智能全球分布式防刷熔断**：请求入境立刻经由局部锁与 KV 锁双重鉴定，一秒内超过 2 次的恶意超频请求在入口处直接物理隔离，彻底保护 Workers 免费额度。
* 📉 **15秒局部内存缓存（极度省额度）**：将黑名单与规则库自动在 Worker 实例中缓存 15 秒，避免高并发下频繁查询 KV。发广播及频道推送走纯官方异步通道，**对 KV 的额外消耗为完美的零次**。
* 🚫 **自由自定义遮罩与脱敏**：采用全新四段式指令。如果第一个参数输入长网址，系统会自动在公开频道中对其执行 HTML 绿色遮罩，只显示你自定义的备注字样（如 `[点击加入技术交流群]`），原始链接完全脱敏。
* 📢 **HTML 级别防爆审计推送卡片**：用户索取节点的瞬间，老板电报会同步收到精致的通知账单。内置 HTML 转义过滤，彻底封杀用户特殊名字字符引起的发信崩溃缺陷。
* 🎛️ **动态身份菜单隔离 (Scope Commands)**：彻底弃用全局菜单。机器人会根据交互者的电报 ID 实时对手机端菜单进行沙箱重构。管理员可见专属管理面板，普通用户完全隐性无感知。

## 🚀 高级进阶——启用全球分布式 KV 空间绑定

### 1. 创建 KV 命名空间
1. 登录 Cloudflare 控制台，进入 **`KV`** 页面。
2. 点击右上角 **`Create a namespace`**，为其命名（例如：`TG_BOT_DB`），点击 **`Add`**。

### 2. 将 KV 绑定至机器人 Worker
1. 进入你机器人的 Worker 管理页面，点击 **`Settings` (设置)** -> 选择左侧的 **`Bindings` (绑定)**。
2. 在 **KV Namespace Bindings** 区域点击 **`Add Binding`**。
3. **严格按照以下规范填写**：
   * **变量名称 (Variable Name)**：必须手动输入大写的 **`TG_LIMIT_KV`**
   * **KV 命名空间 (KV Namespace)**：在下拉菜单中选中刚才创建的 `TG_BOT_DB`。
4. 点击最下方的 **`Save and Deploy`** 即可。

## 🚀 保姆级零瑕疵部署与动态激活

### 步骤 1：让 Bot 成为频道管理员
1. 打开 Telegram，将机器人拉入频道并设置为管理员，保持默认权限开启并点击右下角 **`保存`**。

### 步骤 2：激活 Webhook 通道（仅需执行一次）
将以下链接中的 `<你的BOT_TOKEN>` 和 `<你的Worker独立域名>` 替换为你的真实数据，然后复制到浏览器中访问以激活通道：
```text
[https://api.telegram.org/bot](https://api.telegram.org/bot)<你的BOT_TOKEN>/setWebhook?url=https://<你的Worker独立域名>/

```

### 步骤 3：激活动态菜单显示

1. 部署完成后，老板本人直接前往你的机器人私聊界面，发送一次 `/start` 触发身份探测。
2. 机器人判断你是管理员后，你手机左下角的 `Menu` 菜单会自动重构并加载 `/start`、`/help`、`/manage` 三个完整指令。普通用户端会自动隐藏管理选项。

## 📱 老板专属快捷管理与引流实操说明

管理员直接在私聊框点击左下角菜单中的 `/manage` 即可一键呼出最高权限控制台模板：

### 1. 普通文字节点/文本添加

格式：`添加#关键词#占位备注#节点回复文本`

```text
添加#香港节点#占位#🇭🇰 **香港专线已更新**\n\`vmess://xxxx\`

```

* **频道广播表现**：公网频道会直接干净地发出加粗字体：**香港节点**，完美脱敏。

### 2. 网站链接/群组/广告完美遮罩引流

格式：`添加#真实网址#频道显示的绿色遮罩备注#用户私聊拿到的文本`

```text
添加#[https://t.me/your_qun#](https://t.me/your_qun#)💬 点击加入技术交流群#欢迎加入官方群组交流！

```

* **频道广播表现**：公网频道会隐藏长网址，渲染为精美的超链接：[💬 点击加入技术交流群]，点击即可跳转！

### 3. 高级海报大图广告玩法（图文外链法）

1. 先把你做好的广告海报图片发在任意公开频道或群组，右键选择 **“复制消息链接 (Copy Post Link)”**，得到一个电报官方图片链接（如 `https://t.me/xqkin/123`）。
2. 将此链接作为网址参数输入指令：

```text
添加#[https://t.me/xqkin/123#](https://t.me/xqkin/123#)🖼️ 查看活动海报详情#🎁 欢迎参加特惠活动！优惠码：\`LUCK666\`

```

* **频道广播表现**：频道中会完美显示绿色遮罩文本，且**由于 Telegram 官方的链接预览机制，通知下方会自动展开你那张精美的海报大图预览！** 完美实现零内存消耗的图文广告引流。

### 4. 强力动态黑名单管控

* **一键拉黑用户**：发送 `拉黑#用户数字ID`（例如：`拉黑#6543210`），被拉黑者将瞬间在入口处被全面封锁。
* **一键解除封禁**：发送 `解黑#用户数字ID`（例如：`解黑#6543210`），即可重新释放用户。
